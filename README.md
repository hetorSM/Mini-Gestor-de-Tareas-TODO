# Mini Gestor de Tareas TODO

Aplicación web desarrollada como prueba técnica para la gestión de tareas, utilizando un stack moderno basado en Laravel y herramientas de interactividad en tiempo real.

El objetivo del proyecto es aplicar buenas prácticas de desarrollo, arquitectura limpia y experiencia de usuario eficiente.
aplicar buenas prácticas de desarrollo, arquitectura limpia y experiencia de usuario eficiente.

---

## Características

- Crear tareas (título obligatorio, descripción opcional)
- Editar descripción de tareas
- Marcar y desmarcar tareas como completadas sin recargar la página
- Eliminar tareas con confirmación previa
- Asignar tareas a una categoría (relación uno a muchos)
- Filtrar tareas (todas, pendientes, completadas)
- Interactividad con Alpine.js
- Componentes dinámicos con Livewire
- Diseño responsive con Tailwind CSS

---

## Tecnologías Utilizadas

- **Backend:** Laravel 11+
- **Frontend Reactivo:** Livewire 3+
- **Interactividad:** Alpine.js 3+
- **Estilos:** Tailwind CSS
- **Base de Datos:** MySQL
- **Pruebas:** Pest

---

## Requisitos Previos

- PHP 8.2 o superior
- Composer
- Node.js y NPM
- MySQL o SQLite
